# kclips

A k-means type algorithm for CLustering In the Presence of Scatter

This package implements the k-Clips algorithm for clustering datasets in the presence of scattered, as developed in Maitra and Ramler (2009), where a scattered observation is defined as unlike any other observation in the dataset.

Please cite

Maitra, R. & I. P. Ramler. (2009) Clustering in the presence of scatter. Biometrics, 65:341–352. DOI: 10.1111/j.1541-0420.2008.01064.x. URL: <https://academic.oup.com/biometrics/article/65/2/341/7331779>

in all publications and use of this package.

To install the package, please download or clone and compile using the following at the prompt.

`make run_kclips_estk`

This will create an executable called `run_kclips_estk`

So see how to run the code, please run at the prompt:

`./run_kclips_estk`

This will provide a detailed list of options, along with default values where appropriate, that the program currently has.

Details on methodology are in:

Maitra, R. & I. P. Ramler. (2009) Clustering in the presence of scatter. Biometrics, 65:341–352. DOI: 10.1111/j.1541-0420.2008.01064.x. URL: <https://academic.oup.com/biometrics/article/65/2/341/7331779>
